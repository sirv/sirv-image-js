Installation

npm install sirv-image
const SirvImage = require('sirv-image');

Quick Start

const url = new SirvImage('https://demo.sirv.com/image.jpg')
  .resize({ width: 300, height: 200 })
  .quality(80)
  .format('webp')
  .toUrl();
// https://demo.sirv.com/image.jpg?w=300&h=200&q=80&format=webp

Constructor

new SirvImage(urlOrBase, path?)

ParamTypeDescription
urlOrBasestringFull image URL or base URL
pathstring?Image path (when using base URL)
 
const img1 = new SirvImage('https://demo.sirv.com/image.jpg');
const img2 = new SirvImage('https://demo.sirv.com', '/image.jpg');

Resize

MethodDescription
resize({ width, height, option })Set dimensions and scale option (fit, fill, ignore, noup)
width(w)Set width in pixels
height(h)Set height in pixels
scaleByLongest(s)Resize by longest dimension
thumbnail(size = 256)Create a square thumbnail
 
img.resize({ width: 400, height: 300, option: 'fill' })

Crop

MethodDescription
crop({ width, height, x, y, type, padWidth, padHeight })Crop with optional type and padding
clipPath(name)Apply a Photoshop clipping path
 
img.crop({ width: 200, height: 150, x: 10, y: 20 })
img.crop({ type: 'face', padWidth: 50 })

Rotation

MethodDescription
rotate(degrees)Rotate the image (-180 to 180)
flip()Flip vertically
flop()Flip horizontally

Format & Quality

MethodDescription
format(fmt)Set output format: jpg, png, webp, optimal, original
quality(q)Set JPEG quality (0-100)
webpFallback(fmt)Set WebP fallback format
subsampling(value)Set chroma subsampling: 4:4:4, 4:2:2, 4:2:0
pngOptimize(enabled = true)Enable PNG optimization
gifLossy(level)Set GIF lossy compression (0-100)

hue(v)Adjust hue (-100 to 100)

Color Effects

MethodDescription
colorize({ color, opacity })Apply color overlay
colortone(presetOrOptions)Apply a color tone preset or custom settings
 
img.colortone('sepia')
img.colortone({ color: 'ff5500', level: 80, mode: 'highlights' })

Effects

MethodDescription
blur(v)Apply blur (0-100)
sharpen(v)Apply sharpening (0-100)
vignette({ value, color })Apply vignette effect
opacity(v)Set image opacity (0-100, PNG only)

Text Overlay

text(content, options?)

Add a text overlay. Call multiple times for multiple layers (auto-indexed).

OptionTypeDescription
sizenumberText area width %
fontSizenumberFont size in px
fontFamilystringGoogle Fonts family
fontStylestringnormal, italic
fontWeightstring|number300-800
colorstringText color
opacitynumber0-100
outlineWidthnumberOutline width px
outlineColorstringOutline color
backgroundColorstringBackground color
backgroundOpacitynumberBackground opacity
alignstringleft, center, right
positionstringcenter, north, south, etc.
positionXstring|numberX offset
positionYstring|numberY offset
 
img.text('Hello', { color: 'white', fontSize: 24, position: 'center' })
   .text('World', { position: 'south' }) // second text layer

Watermark

watermark(path, options?)

Add a watermark overlay. Call multiple times for multiple watermarks.

OptionTypeDescription
positionstringcenter, north, tile, etc.
positionXstring|numberX offset
positionYstring|numberY offset
scaleWidthstring|numberWidth (px or %)
scaleHeightstring|numberHeight (px or %)
scaleOptionstringnoup, fit, fill, ignore
rotatenumber-180 to 180
opacitynumber0-100
layerstringfront, back
 
img.watermark('/wm.png', { position: 'center', opacity: 50 })

Canvas

canvas({ width, height, color, position, opacity, aspectRatio, borderWidth, borderHeight, borderColor, borderOpacity })

Configure canvas settings.

img.canvas({ width: 800, height: 600, color: 'ffffff' })
img.canvas({ aspectRatio: '16:9' })
img.canvas({ borderWidth: 10, borderColor: '000000' })

Frame

frame({ style, color, width, rimColor, rimWidth })

Add a frame. Styles: solid, mirror, edge, dither, none.

img.frame({ style: 'solid', color: '333333', width: 5 })

Other

MethodDescription
page(num)Select a PDF page to render
profile(name)Apply a saved transformation profile

URL Generation

MethodDescription
toUrl()Build and return the final URL string
toString()Alias for toUrl(). Enables template literal usage
 
const url = img.toUrl();
console.log(`Image: ${img}`);

Full Examples

E-commerce Product Image

const url = new SirvImage('https://demo.sirv.com', '/product.jpg')
  .resize({ width: 800, option: 'fill' })
  .crop({ type: 'face' })
  .format('webp')
  .quality(85)
  .sharpen(20)
  .watermark('/logo.png', { position: 'southeast', opacity: 40 })
  .text('Sale!', { color: 'red', fontSize: 32, position: 'north' })
  .toUrl();

Vintage Photo Effect

const url = new SirvImage('https://demo.sirv.com/photo.jpg')
  .resize({ width: 600 })
  .colortone('sepia')
  .contrast(10)
  .vignette({ value: 40 })
  .frame({ style: 'solid', color: '333', width: 8 })
  .quality(90)
  .toUrl();
